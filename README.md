# MicroPython Programming â€“  Learning Materials

This repository contains structured **MicroPython learning materials** designed for beginners and Certificate III/IV ICT students.  
The content is delivered using **Jupyter Notebooks (.ipynb)** and focuses on **hands-on programming concepts** commonly used in **IoT and embedded systems**.

These materials are suitable for classroom delivery, self-paced learning, and assessment preparation.

---

##  What You Will Learn

Students will progressively develop skills in:

- MicroPython fundamentals
- Variables and data types
- Input, sequence, and selection
- Conditional logic (`if`, `elif`, `else`)
- Nested decision making
- Lists, tuples, and dictionaries
- File handling and data storage
- Debugging using `assert`
- Networking fundamentals for IoT
- MQTT communication
- Cloud integration using Supabase
- Problem-solving through practical and assessment-style tasks
